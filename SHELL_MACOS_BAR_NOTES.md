# ClariceOS-Theme RC6 — macOS-inspired continuous top bar

This revision remodels the GNOME Shell top bar to feel closer to a continuous macOS menu bar.

## What changed
- Replaced pill-heavy panel buttons with a continuous panel surface
- Removed default panel button borders and shadows
- Added subtle hover and active states that sit inside the bar instead of floating above it
- Gave the clock a slightly elevated treatment so it reads as the visual anchor of the bar
- Applied the redesign to both light and dark variants

## Scope
This package focuses on the GNOME Shell top bar only. GTK files were preserved from the previous complete package.
