# GNOME Theme Visual Package v3

Pacote visual revisado para implementação de tema GNOME, fiel ao mockup aprovado e com os assets de interface sem texto embutido.

## Estrutura
- `assets/svg/` — componentes vetoriais prontos para edição
- `assets/png/` — renders PNG dos mesmos componentes
- `mockups_clean/svg/` — mockups de janela sem texto, para referência visual e layout
- `mockups_clean/png/` — renders PNG dos mockups limpos
- `reference/` — mockup original aprovado e grade de medidas
- `tokens/` — paleta e dimensões

## Importante
Os arquivos usados para implementação ficam em `assets/` e `mockups_clean/`.
O diretório `reference/` mantém apenas a referência visual original.
