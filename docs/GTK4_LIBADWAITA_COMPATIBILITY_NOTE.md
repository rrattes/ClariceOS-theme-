# ClariceOS-Theme — GTK4 / libadwaita Compatibility Note

ClariceOS-Theme targets current GNOME with an honest scope:

- It provides a GTK4 theme layer and visual tokens aligned to the approved mockup.
- It is designed to work alongside the GTK4 symlink method already shipped in `scripts/apply-gtk4-symlink.sh`.
- It aims for visual coherence in modern GNOME apps, not a guarantee that every libadwaita app will be fully re-themed in every environment.

This is the correct distribution stance for a GNOME-current theme: practical compatibility, documented setup, and no promise of universal replacement of Adwaita internals.
