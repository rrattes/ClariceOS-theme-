# RC7 Quick Settings refinement

This release refines the GNOME Shell control box / Quick Settings for both dark and light variants.

## What changed
- stronger surface hierarchy for the Quick Settings popover
- separate treatment for the popover shell, icon buttons, tiles and sliders
- reduced plain-white feeling in the light theme
- richer gradients and stronger internal depth in the dark theme
- improved subtitle legibility and checked-state polish
- softer, more premium separators and spacing

## Scope
This package keeps the macOS-inspired continuous top bar from RC6 and focuses only on refining the Shell control box.
